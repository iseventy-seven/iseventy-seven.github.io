条件运算符

  • 比较符号
    • > 大于
    • < 小于
    • >= 大于等于
    • <= 小于等于

    • != 不等于

第四讲 数组

一维数组

  • 定义

    • 类型 变量名[数组长度];
    • 初始化
      • int a[3] = {0, 1, 2};
      • int b[] = {0, 1};
      • int c[5] = {1};
      • int d[10] = {0};
    • 大数组定义在函数外（堆空间），其值为 0

  • 调用

    • 通过下标访问，下标为序号减一

多维数组

  • 数组的每一个值都是一个数组
  • 定义
    • 类型 变量名[数组长度][数组长度];
    • 初始化
      • memset(a, 0, n * sizeof(int));

第五讲 字符串

  • ASCII 码
    • 0：48
    • a：65
    • A：97
    • 可参与运算
  • 字符数组
    • 字符数组末尾加 \0 即为字符串
    • %s
    • cin、scanf("%s", a)、getline()
    • fgets(s, 100, stdin) 到数组，不过滤回车
    • cin.getline(s, len) 到数组
    • getline(cin, s) 到字符串
    • 读入遇到空格、回车、结束符为止
  • 字符数组常用函数
    • strlen 长度
    • strcmp 字典序
    • strcpy 字符串赋值，后给前
  • string
    • printf("%s\n", str.c_str());
    • str.empty();
    • str.size();
    • 相加（自动转换为字符串对象，对于数值，会自动转化为字符）、比较、赋值
    • for (char c: str) cout << c; 遍历
    • for (char &c: str) c = 'a'; 遍历修改（引用变量）
    • str.pop_back(); 删除最后一个元素
    • str.back(); 返回最后一个元素
    • substr(i, len)
