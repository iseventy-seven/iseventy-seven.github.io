1.1 Welcome 欢迎

1.2 What is Machine Learning 机器学习是什么

  • 在特定编程的情况下，基于计算机学习能力的领域。

  • 一个程序被认为能从经验 $E$ 中学习，解决任务 $T$，达到性能度量值 $P$，当且仅当，有了经验 $E$ 后，经过 $P$ 评判，程序在处理 $T$ 时的性能有所提升。

1.3 Supervised Learning 监督学习

  • 监督学习是指给学习算法一个数据集，这个数据集由“正确答案”组成，然后运用学习算法，算出更多的正确答案。

2.2 代价函数 Cost Function

  • 回归问题的本质就是求出假设函数 $h_\theta(x)$（又作 $h(x)$），因此需要确定参数（Parameters）$\theta_0$ 和 $\theta_1$。

  • 对于 $h_{\theta}(x) = \theta_0 + \theta_1 x$

    • 建模误差（Modeling Error）：$h_{\theta}(x^{(i)}) - y^{(i)}$

      • 表示模型所预测的值与训练集中的实际值之间的差距。

    • 代价函数（Cost Function/Squared Error Function 平方误差函数）：$J(\theta_0, \theta_1)$

      • 定义在整个训练集上，表示所有样本误差的平均。

    • 选择参数使得建模误差的平方和最小，即使得代价函数

2.5 梯度下降 Gradient Descent

  • 梯度下降算法

    • 目的：求函数最小值

    • 思想：开始时选择一个随机参数的组合 $(\theta_0, \theta_1, \cdots, \theta_n)$，计算代价函数，然后寻找下一个能让代价函数值下降最多的参数组合。反复上面步骤直到找到一个局部最小值（Local Minimum），由于没有遍历所有的函数组合，不能确定得到的局部最小值是否是全局最小值（Global Minimum）。

2.6 梯度下降的直观理解 Gradient Descent Intuition

  • 一元变量下的梯度下降过程就是不断的通过减一个值或加一个值，使得 $\theta$ 逐渐接近局部最优解。而当 $\alpha$ 值合理时，由于偏导数正负的不同，梯度下降会自动向最优解方向进行。

  • 二元变量下的梯度下降过程就是不断寻找下降最快的“路”，不同的初始值可能导致不同的局部最优解。

  • $\alpha$ 值过小或过大，会导致 $\theta$ 值下降过慢或者不能收敛。

  • 初始点为局部最优点，偏导数为 $0$，不会再下降。

  • 随着梯度下降的进行，其变化幅度由于偏导数值的变化，会自动变小。

2.7 线性回归的梯度下降 Gradient Descent For Linear Regression

  • 梯度下降与线性回归

    • 偏导

    • 算法

    • 凸函数：一个弓形函数，没有局部最优解，只有一个全局最优解。对于线性回归的代价函数，是一个凸函数，即总会收敛到一个局部最优解。

    • 批量梯度下降（Batch Gradient Descent）：每一步下降都遍历整个训练集的样本。

    • 在数据量较大的情况下，相比正规方程组（Normal Equations）的方法，梯度下降法更加适用。

3.1 矩阵和向量 Matrices and Vectors

  • 矩阵 Matrix

    • 矩阵的维数（Dimension of Matrix）：$raws \times columns$

  • 向量 Vector：一个 $n \times 1$ 的矩阵

3.2 加法和标量乘法 Addition and Scalar Multiplication

  • 矩阵加法：维度相等时对应相加
  • 矩阵标乘：每个元素分别相乘

3.3 矩阵向量乘法 Matrix Vector Multiplication

  • $m \times n$ 的矩阵乘以 $n \times 1$ 的向量，得到 $m \times 1$ 的向量。

3.6 逆、转置 Inverse and Transpose

  • 矩阵的逆：设矩阵 $A$ 是一个 $m \times m$ 的方阵，如果有逆矩阵，则：$AA^{-1} = A^{-1}A = I$。

    • 不存在逆矩阵的矩阵叫奇异（Singular）矩阵或退化（Degenerate）矩阵。

  • 矩阵的转置：设 $A$ 为 $m \times n$ 阶矩阵，第 $i$ 行 $j$ 列的元素是 $a(i, j)$，即：$A = a(i, j)$。定义 $A$ 的转置为一个 $n \times m$ 阶矩阵 $B$，满足 $B = a(j, i)$，即 $b(i, j) = a(j, i)$，其中 $B$ 的第 $i$ 行第 $j$ 列元素是 $A$ 的第 $j$ 行第 $i$ 列元素，记 $A^T = B$ 或 $A’ = B$。
