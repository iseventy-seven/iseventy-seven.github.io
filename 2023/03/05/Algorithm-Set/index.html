函数技巧

  • swap

    • 头文件：<algorithm>
    • 功能：交换两个变量的值
    • 调用方法：swap(a, b)
      • a\b 为待交换变量

  • reverse

    • 头文件：<algorithm>
    • 功能：翻转数组
    • 调用方法：reverse(a, a + k)
      • a\a + k 分别表示数列待翻转部分的起始位置和终止位置的下一位。

  • memset

    • 头文件：<cstring>
    • 功能：为某一块空间赋值
    • 调用方法：memset(a, x, n * sizeof(type))
      • a 表示待赋值空间，一般为一个数组。
      • x 为所赋的值，由于 memset 函数以字节为单位赋值，通常仅取 x 为 0 或 -1，其二进制分别为 0000 0000 和 1111 1111。
      • n 为待赋值类型的数量，sizeof(type) 表示待赋值类型的单位字节数，二者乘积表示待赋值空间的字节大小。

  • memcpy

    • 头文件：<cstring>
    • 功能：复制空间的值
    • 调用方法：memcpy(b, a, sizeof(a))
      • b 目标空间
      • a 原空间
